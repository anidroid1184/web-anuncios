**INSTRUCCIONES DE ANÁLISIS:**

Para cada activo visual (imagen o frame de video):

1. **Visual Forensics (100 palabras)**:
   - Composición: regla de tercios, jerarquía visual
   - Paleta de colores dominantes y contraste
   - Elementos clave: sujetos, texto, logos, CTAs
   - Calidad técnica: resolución, iluminación

2. **Semiotic Analysis (120 palabras)**:
   - Significados literales vs emocionales
   - Símbolos y códigos culturales
   - Arquetipo de marca (héroe, sabio, rebelde, etc.)
   - Narrativa visual implícita
   - Posicionamiento percibido (premium, accesible, disruptivo)

3. **Psychological Triggers (100 palabras)**:
   Identifica qué gatillos se activan:
   - Escasez (urgencia, tiempo limitado)
   - Prueba social (testimonios, números de usuarios)
   - Autoridad (expertos, certificaciones)
   - Reciprocidad (valor gratuito, regalos)
   - Pertenencia (identificación tribal/comunitaria)
   - Contraste (antes/después, comparaciones)
   - FOMO (aversión a la pérdida)

4. **Effectiveness Scores (1-10 con justificación breve)**:
   - Stopping Power: ¿Detiene el scroll?
   - Message Clarity: ¿Se entiende en 3 segundos?
   - Emotional Relevance: ¿Conecta emocionalmente?
   - CTA Strength: ¿Call to action claro y urgente?
   - Brand Recall: ¿Se recuerda la marca?

5. **Optimization Roadmap (3-5 acciones específicas)**:
   - Qué cambiar exactamente (no generalidades)
   - Por qué mejorará el rendimiento
   - Prioridad (ALTA/MEDIA/BAJA)
   - Impacto esperado en métricas

**REGLAS CRÍTICAS:**
- Genera SOLO JSON, sin texto adicional antes o después
- Asegúrate de cerrar todas las llaves y corchetes correctamente
- No uses comillas dobles dentro de strings sin escapar
- Mantén el formato compacto pero completo
- Si hay muchos activos (>10), prioriza calidad sobre cantidad en cada análisis individual
